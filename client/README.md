# Folio — Client

React 19 + TypeScript + Vite frontend for the Folio bookstore.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build outputs to `dist/` and is served as static files by the Express backend.

## Stack

- React 19, TypeScript, Vite
- Tailwind CSS + CSS Modules
- React Router v6
- Stripe.js (payment UI)
- Framer Motion (animations)
